This is the official Vouch - Release on delivery payment gateway plugin for WooCommerce. Allows you to accept credit cards, debit cards, netbanking, wallet, and UPI payments with the WooCommerce plugin. It uses a seamles integration, allowing the customer to pay on your website This allows for refunds, works across all browsers, and is compatible with the latest WooCommerce.

== Configuration ==

1. Visit the WooCommerce settings page, and click on the Payment tab.
2. Make sure you have activated the plugin in the WordPress Plugin Manager
3. Click on Vouch to edit the settings.
4. Add the Api key that you receive from vouch.
5. Add Your Online Store name field(this is the common prefix for all the vouch gigs from your site).
4. Click on save changes(make sure that you add the api key and your store name fields as they are mandatory)
